## Time logger

### Installation

```
npm install -g https://github.com/nishanBende/time-logger
```

### Start Logging

```
log-time
```

#### Optional flags

```
log-time -p [projectname]
```

`Note: Timesheet files are saved in HOME_DIR/time-logs/[Month]-[Year].json`
